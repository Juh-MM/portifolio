Portfólio web site

  Após uma breve pesquisa no mercado de trabalho, percebi a importância de ter um espaço dedicado para apresentar meus projetos e compartilhar os aprendizados que venho acumulando ao longo da minha trajetória. Desenvolver este portfólio também foi uma ótima oportunidade para explorar o Tailwind CSS e aprender mais sobre a suas classes utilitárias na prática.

PinCat web site

  O PinCat foi um projeto desenvolvido durante a faculdade, no qual fui desafiada a criar uma aplicação Fullstack completa de forma totalmente independente. Nesse processo, aprendi a transformar um projeto simples em um PWA, além de ter meu primeiro contato com back-end, utilizando Node.js e MongoDB.

Salão senac web site

  O Salão Senac foi um projeto em grupo idealizado pela faculdade, com o objetivo de digitalizar e otimizar o agendamento de serviços da empresa. Durante o desenvolvimento, aprendi muito sobre versionamento de código, trabalho em equipe e o uso do Bootstrap para criação de interfaces com mais agilidade.

Sobre mim

  Olá! Meu nome é Júlia Martins, sou apaixonada por design e desenvolvimento web, áreas onde consigo unir criatividade, estética e funcionalidade para transformar ideias em soluções digitais eficientes, sempre focando na melhor experiência para o usuário. Atualmente, estou cursando Análise e Desenvolvimento de Sistemas (ADS) e Design.

  Tenho experiência com tecnologias como React, Tailwind CSS, JavaScript, Node.js, CSS3, HTML5 e Figma. Se você gostou do meu trabalho ou tem alguma ideia para tirar do papel, ficarei muito feliz em conversar. Entre em contato e vamos construir algo incrível juntos!
